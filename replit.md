# Overview
This project is a Bee Swarm Simulator utility application, providing tools like stats tracking and promotional codes.

- Verified application functionality

# User Preferences
- Preferred language: English
- Keep `tsx` in dependencies (not just devDependencies) to avoid issues on restarts

# Project Architecture
- Frontend: React with Vite, Tailwind CSS, Shadcn UI, Framer Motion
- Backend: Express server using `tsx` for execution
- Routing: `wouter` for frontend, Express for API
- Data: TanStack Query for state management and fetching
